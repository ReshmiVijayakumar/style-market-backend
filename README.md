# style-market-backend
Style Market Backend — A production-ready RESTful API for an e-commerce platform, built with Node.js, Express, TypeScript, and PostgreSQL. Supports user authentication, product catalog, cart &amp; wishlist management, and secure checkout with payments.

## 🚀 Features
- User Authentication (JWT)
- Product listing, details & search
- Cart & Wishlist APIs
- Checkout & Orders
- Payment gateway integration (Stripe / Razorpay)
- API documentation with Swagger
- Input validation (Zod)
- Centralized error handling
- Docker + Docker Compose setup
- Prisma ORM + PostgreSQL
- Redis caching (optional for sessions/cart)

---

## 🏗️ Tech Stack
- **Backend Framework**: Express.js (Node.js + TypeScript)
- **Database**: PostgreSQL (via Prisma ORM)
- **Authentication**: JWT + bcrypt
- **Cache/Queue**: Redis (optional)
- **Validation**: Zod
- **Documentation**: Swagger
- **Testing**: Jest + Supertest
- **Logging**: Pino
- **Deployment**: Docker + Docker Compose
